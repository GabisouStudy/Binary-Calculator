== QUICK START ==

Run the start script for your operating system.

Windows: start_windows.bat
Mac:     start_mac.command
Linux:   start_linux.sh

Close the console or press Control+C to stop the server.

== CONFIGURATION ==

1. Add your Ruby scripts to the PortableRuby directory.
2. Modify the start files to point to your Ruby script.

Create your own startup files to run multiple Ruby scripts!
